TCGA Cancer Subtype Classification Using Machine LearningThis project investigates the use of Machine Learning methods for multiclass cancer subtype classification using high-dimensional gene expression data from The Cancer Genome Atlas (TCGA). The goal is to compare baseline, filter-based, embedded, and wrapper-style feature selection approaches for reducing dimensionality while maintaining strong subtype classification performance.The project evaluates multiple machine learning pipelines across three TCGA cancer datasets:BRCA — Breast Invasive CarcinomaCOAD — Colon AdenocarcinomaPRAD — Prostate AdenocarcinomaThe datasets contain approximately 60,660 gene expression features per sample, making dimensionality reduction and feature selection critical components of the workflow.Project GoalsPerform multiclass cancer subtype classification using gene expression data.Compare different feature selection strategies (Filter, Wrapper, Embedded).Evaluate dimensionality reduction effectiveness and the "Baseline Paradox."Analyze tradeoffs between runtime, interpretability, and predictive performance.Investigate how aggressive feature reduction affects classification quality in imbalanced datasets.DatasetsThe project uses TCGA gene expression matrices and associated subtype labels.BRCA: 1,231 samples, 5 subtypes.COAD: 524 samples, 4 subtypes.PRAD: 554 samples, 8 subtypes (high class imbalance).Each dataset includes a gene expression matrix and aligned patient subtype labels.Project StructureBaseline/: Scripts for training on the full 60,660 feature set.Filter/: Statistical methods (Variance Threshold, ANOVA/F-test, Mutual Information).Embedded/: Model-integrated selection (LASSO Logistic Regression).Wrapper/: Search-based selection (SelectKBest, Forward Selection with pre-filtering).data/: Local directory for TCGA gene expression datasets and labels.results/: Centralized output for plots, CSV summaries, and performance visualizations.How to Reproduce ResultsTo replicate the benchmarks and visualizations found in the final report, follow these steps:Environment Setup:Ensure you have Python 3.9+ installed, then install dependencies:Bashpip install pandas numpy scikit-learn matplotlib
Data Placement:Place the TCGA parquet/csv files in the data/ directory.Run Benchmarks:Execute the following scripts in order to generate results for each category:Bashpython Baseline/baseline_run.py
python Filter/run_filters.py
python Wrapper/select_k_best.py
python Embedded/lasso_experiments.py
Generate Visualizations:Run the plotting utility to generate the comparison charts used in the report:Bashpython results/generate_comparison_plots.py
Methodology1. Baseline ModelsEstablished using the full feature set ($p=60,660$) with a Logistic Regression SAGA solver to create a performance "ceiling" for comparison.2. Filter MethodsEvaluates features independently of the classifier.Variance Threshold: Tested sweeps from 0.05 to 0.75.ANOVA / F-Test: Ranks genes by statistical significance.Mutual Information: Captures non-linear relationships.3. Wrapper MethodsUses a Variance Pre-filter to reduce the search space to 2,000 genes to manage computational overhead, followed by:SelectKBest: Selection based on top-performing feature subsets.Forward Selection: Iterative addition of predictive features.4. Embedded MethodsLASSO (L1 Regularization): Forces non-informative coefficients to zero during training.Random Forest Importance: Uses tree-based impurity reduction to rank genes.Evaluation MetricsMetricPurposeAccuracyOverall classification correctness.Macro F1Performance across all classes equally (Primary metric for PRAD).ROC-AUCMulticlass discrimination capability.RuntimeMeasure of computational efficiency and scalability.Key FindingsThe Baseline Paradox: The full-feature baseline often provides a performance ceiling that aggressive selection cannot surpass.Efficiency: Filter methods like ANOVA achieve 90%+ dimensionality reduction with minimal performance loss.Class Imbalance: PRAD remains the most challenging dataset; while accuracy is high, Macro F1 plateaus due to minority subtype neglect.AuthorsWilliam Alex Ross, Qianhe (Ted) Sha, Bryan Pham, Yassin Lahrime.
